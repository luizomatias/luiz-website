# Luiz Otávio — Portfolio

A one-page portfolio for Luiz Otávio Matias da Luz, Applied ML Lead.

The centerpiece is a portrait that switches the whole site between two
worlds — **IRL** (paper-and-ink editorial) and **anime** (sky and halftone) —
with light and dark themes layered on top. Built with a circular world-wipe
transition, ambient sakura/firefly particles, and katakana decode effects.

## Tech

Vite · TypeScript · hand-written CSS (no framework)

## Getting started

```bash
npm install
npm run dev      # start the dev server
npm run build    # build to dist/
npm run preview  # preview the production build
```
